Student Assignment Tracker

A Java Swing desktop application for managing courses, assignments, student submissions, grading, and academic progress.

View Source Code Run Locally

Project Overview

Student Assignment Tracker helps teachers and students manage academic assignment workflows. Teachers can create courses and assignments, review submissions, grade student work, and provide feedback. Students can view assignments, submit work, and track their progress.

Key Features

Teacher Features

  • Create and manage courses
  • Create assignments with due dates
  • Review student submissions
  • Grade submissions and give feedback

Student Features

  • Register and log in as a student
  • View assigned coursework
  • Submit assignment work
  • Check grades and feedback

System Features

  • Role-based login system
  • Java Swing user interface
  • JSON-based data storage
  • Object-oriented project structure

Technology Stack

Java Maven Java Swing Jackson JSON OOP

Run Locally

This is a desktop Java application. Clone the repository and run it with Maven:


git clone https://github.com/mdjunayet10/Student-Assignment-Tracker.git
cd Student-Assignment-Tracker
mvn clean compile
mvn exec:java -Dexec.mainClass="org.app.Main"
