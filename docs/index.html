Run the desktop app locally

This GitHub Pages site is a professional landing page. The real application is a Java Swing desktop app. 

git clone https://github.com/mdjunayet10/Student-Assignment-Tracker.git
cd Student-Assignment-Tracker
mvn clean compile
mvn exec:java -Dexec.mainClass="org.app.Main"
