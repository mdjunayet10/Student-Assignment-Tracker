Run Locally

Clone the repository, build it with Maven, and run the Java application:

git clone https://github.com/mdjunayet10/Student-Assignment-Tracker.git
cd Student-Assignment-Tracker
mvn clean compile
mvn exec:java -Dexec.mainClass="org.app.Main"
