Student Assignment Tracker

A Java-based assignment management system for teachers and students to manage courses, assignments, submissions, grading, and progress tracking.

View on GitHub Run Locally

Project Overview

Student Assignment Tracker helps teachers create courses and assignments, while students can view assignments, submit work, and track grades. The project follows Object-Oriented Programming principles using Java, Maven, Swing GUI, service classes, models, and JSON-based data persistence.

Features

Teacher Tools

  • Create and manage courses
  • Create assignments
  • Review student submissions
  • Grade work and give feedback

Student Tools

  • View enrolled courses
  • See active assignments
  • Submit assignment content
  • Track grades and feedback

System Features

  • Role-based login
  • JSON data persistence
  • Java Swing interface
  • Maven project structure

Technology Stack

  • Java 23
  • Maven
  • Java Swing GUI
  • Jackson JSON processing
  • FlatLaf look and feel

Run Locally

Clone the repository, build it with Maven, and run the Java application:

git clone https://github.com/mdjunayet10/Student-Assignment-Tracker.git
cd Student-Assignment-Tracker
mvn clean compile
mvn exec:java -Dexec.mainClass="org.app.Main"
